## Introduction

This application is a basic To-Do list that allows users to create, read, update, and delete tasks. It was developed using TypeScript for the logic and Tailwind CSS for the styling. The project provides a clean and responsive user interface, making it easy for users to manage their tasks.

## Features

- Create a new task with a title and description.
- Mark tasks as complete or incomplete.
- Edit existing tasks to update their details.
- Delete tasks that are no longer needed.
- Filter tasks based on their completion status.
- Responsive design for optimal user experience across different devices.
